Invoker Commands Playground

Control dialogs, popovers, and media with pure HTML attributes. No JavaScript required.

Baseline 2025 — command / commandfor

Your browser does not support the Invoker Commands API.

Try Chrome 135+, Edge 135+, or Firefox 138+. The examples below still show the markup but interactive demos may not work.

1 Dialog Controls

Open and close <dialog> elements using command="show-modal" and command="close" — zero JavaScript.

Show Modal / Close Dialog
<!-- The button references the dialog by id -->
<button commandfor="myDialog" command="show-modal">
  Open Dialog
</button>

<dialog id="myDialog">
  <h3>Hello from a Modal</h3>
  <p>Opened with command="show-modal". No JS!</p>
  <button commandfor="myDialog" command="close">Close</button>
</dialog>
Live Demo

Request Close (Cancelable)
<button commandfor="confirmDialog" command="show-modal">
  Delete Item
</button>

<dialog id="confirmDialog">
  <h3>Confirm Deletion</h3>
  <p>Are you sure?</p>
  <!-- request-close fires a "close" event that can be canceled -->
  <button commandfor="confirmDialog" command="request-close">
    Cancel
  </button>
  <button commandfor="confirmDialog" command="close">
    Confirm
  </button>
</dialog>

2 Popover Controls

Toggle, show, and hide popovers with command="toggle-popover", command="show-popover", and command="hide-popover".

Toggle / Show / Hide Popover
<button commandfor="info" command="toggle-popover">Toggle</button>
<button commandfor="info" command="show-popover">Show</button>
<button commandfor="info" command="hide-popover">Hide</button>

<div id="info" popover>
  <h4>Popover Content</h4>
  <p>Controlled entirely via HTML attributes.</p>
</div>
Live Demo

Popover Content

This popover is controlled entirely via HTML attributes. No JavaScript event handlers needed.

Manual Popover (Explicit Dismiss Only)
<button commandfor="sticky" command="show-popover">Show</button>
<!-- popover="manual" requires explicit hide; clicking outside won't close it -->
<div id="sticky" popover="manual">
  <p>I won't close when you click outside!</p>
  <button commandfor="sticky" command="hide-popover">Dismiss</button>
</div>
Live Demo

Sticky Popover

3 Custom Commands

Define custom commands prefixed with --. The target element receives a command event you can handle in JavaScript.

Custom Command with Event Listener
<button commandfor="widget" command="--reset">Reset</button>
<button commandfor="widget" command="--increment">+1</button>
<button commandfor="widget" command="--decrement">-1</button>

<div id="widget">Count: 0</div>

<script>
  const widget = document.getElementById('widget');
  let count = 0;
  widget.addEventListener('command', (e) => {
    if (e.command === '--increment') count++;
    if (e.command === '--decrement') count--;
    if (e.command === '--reset') count = 0;
    widget.textContent = `Count: ${count}`;
  });
</script>

4 Media Controls

5 Old Way vs New Way

See how much boilerplate Invoker Commands eliminate compared to traditional JavaScript patterns.

Old Way (JavaScript)
<!-- HTML -->
<button id="openBtn">Open</button>
<dialog id="dlg">
  <p>Content</p>
  <button id="closeBtn">Close</button>
</dialog>

<!-- JavaScript -->
<script>
  const dlg = document
    .getElementById('dlg');
  const openBtn = document
    .getElementById('openBtn');
  const closeBtn = document
    .getElementById('closeBtn');

  openBtn.addEventListener(
    'click', () => {
      dlg.showModal();
  });

  closeBtn.addEventListener(
    'click', () => {
      dlg.close();
  });
</script>
~20 lines · HTML + JavaScript required
New Way (Invoker Commands)
<!-- HTML only. No JavaScript. -->
<button
  commandfor="dlg"
  command="show-modal">
  Open
</button>

<dialog id="dlg">
  <p>Content</p>
  <button
    commandfor="dlg"
    command="close">
    Close
  </button>
</dialog>



<!-- That's it. Zero JavaScript. -->
~12 lines · Pure HTML · 40% less code
Old Way: Popover Toggle
<button id="togBtn">Toggle</button>
<div id="pop" popover>...</div>

<script>
  document.getElementById('togBtn')
    .addEventListener('click',
      () => {
        document
          .getElementById('pop')
          .togglePopover();
      });
</script>
~10 lines · JS boilerplate
New Way: Popover Toggle
<button
  commandfor="pop"
  command="toggle-popover">
  Toggle
</button>

<div id="pop" popover>
  ...
</div>


<!-- Done. Declarative. -->
~6 lines · Pure HTML · 40% less code

8 Quick Reference

All built-in command values at a glance.

Built-in Commands
Command Target Equivalent JS
show-modal <dialog> dialog.showModal()
close <dialog> dialog.close()
request-close <dialog> dialog.requestClose()
show-popover [popover] el.showPopover()
hide-popover [popover] el.hidePopover()
toggle-popover [popover] el.togglePopover()
play-media <video> / <audio> media.play()
pause-media <video> / <audio> media.pause()
toggle-muted <video> / <audio> media.muted = !media.muted
--custom-name Any element el.addEventListener('command', ...)
